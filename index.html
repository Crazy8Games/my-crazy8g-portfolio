Pokemon-EM-RB-Css

Pokemon-EM-RB-Css is a pokemon emerald/ruby styling framework. To simply add the main css just copy the code below then add it to any of your css files. Then copy the link tag in any html you choose.

Poke-EM-RB-Css Code


/* Variables */
:root {
  --primary-color: #78c850; /* Emerald's primary color */
  --secondary-color: #f08030; /* Ruby's primary color */
  --text-color: #000; /* Black text color */
  --background-color: #fff; /* White background color */
}

/* Base Styles */
html,
body {
  height: 100%;
}

body {
  margin: 0;
  padding: 0;
  font-family: "Pokemon Emerald", "Pokemon Ruby", sans-serif;
  font-size: 16px;
  color: var(--text-color);
  background-color: var(--background-color);
}

/* Header */
.header {
  background-color: var(--primary-color);
  color: var(--text-color);
  padding: 10px;
  text-align: center;
}

/* Buttons */
.button {
  display: inline-block;
  padding: 10px 20px;
  background-color: var(--secondary-color);
  color: var(--text-color);
  border: none;
  cursor: pointer;
  text-align: center;
  text-decoration: none;
  border-radius: 5px;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: darken(var(--secondary-color), 10%);
}

/* Cards */
.card {
  background-color: var(--background-color);
  box-shadow: 0px 0px 10px rgba(0, 0, 0, 0.1);
  border-radius: 5px;
  padding: 20px;
  margin: 20px;
}

.card-highlight {
  background-color: var(--primary-color);
  color: var(--text-color);
}
